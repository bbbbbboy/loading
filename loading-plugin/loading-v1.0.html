写这个插件的初衷

$.ajax({
    type: "post",
    url: "/xxx/xxx",
    data: {a:b,c:d},
    beforeSend: function () {
        //显示遮罩层
        $('body').loading({
            uid: 'loading1',
            iconUrl: 'images/loading_2.gif',
            position: 'fixed'
        });
    },
    success: function (result) {

    },
    compelete: function () {
        //隐藏遮罩层
        $('body').loading('hide');
    },
    error: function () {

    }
});

1.考虑到 jquery ajax 在网络卡的时候
或者请求大量数据的时候
数据返回的间隔时间会很长
这样子渲染页面就会有个卡顿感
用 loading 可以提升用户体验
起到一个平滑的作用

2.考虑到 点击一个按钮后 在网络卡的时候的等待

整个屏幕的遮罩

 $('body').loading({
    uid: 'loading1',
    iconUrl: 'images/loading_2.gif',
    position: 'fixed'
 });

body最简单的配置[ 简单点,配置的方式简单点 ]
uid：当前loading的唯一标识
iconUrl：loading图片
position：定位方式[ $('body') 要用fixed,不要问为什么 : ) ]

运行 显示[需要运行后,无销毁] 关闭 销毁

局部的遮罩

 $('#demoBtn2').loading({
    uid: 'loading2',
    iconUrl: 'images/loading_2.gif',
    position: 'absolute',
    width:37/2,
    height:37/2
 });

局部的配置
uid：当前loading的唯一标识
iconUrl：loading图片
position：定位方式[ 用absolute,不要问为什么 : ) ]
width：手动设置图片的宽
height：手动设置图片的高

运行[我的id为demoBtn2] 显示[需要运行后,无销毁] 关闭 销毁

默认参数配置

$(function () {

    $.loading.setDefaults({
       iconUrl: 'images/loading_2.gif',
       opacity: 0.7,
       width:37,
       height:37,
       fade:300
    });

});

用来配置loading的默认参数
实际调用的时候,就可以少配置这些参数咯 : )

相关操作

$('xxx').loading("show");

$('xxx').loading("hide");

$('xxx').loading("destroy");

xxx为类名
"show"：显示loading,前提是有运行过的loading
"hide"：隐藏loading
"destroy"：销毁loading

参数列表

参数 参数类型 默认参数 参数说明
uid String ''

唯一标识[*必须配置]

唯一标识区分于页面其他loading

同一页面上不能重复
position String ''

定位方式[*必须配置]

作用在 body 要用 fixed

作用在局部 用absolute
iconUrl String ''

gif图片[*必须配置]

loading的gif图

自定义图片,要结合loading的透明背景,不然会影响美观
opacity Number 0.7

透明度

透明度 0.0~1.0
width Number ''

gif 图片的宽

width 可以不用配置,默认原图的宽
height Number ''

gif 图片的高

height 可以不用配置,默认原图的高
fade Number ''

淡入淡出效果

fade 可以不用配置,默认没有淡入淡出效果
